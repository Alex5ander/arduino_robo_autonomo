#### Lista de materiais utilizados

* 1 Driver Motor Ponte H L298n.
* 1 Sensor de Distância Ultrassônico HC-SR04.
* 1 Suporte Sensor Ultrassônico HC-SR04.
* 1 Suporte de baterias 18650
* 2 baterias 18650 de 4.2 volts cada
* 1 Arduino Uno R3
